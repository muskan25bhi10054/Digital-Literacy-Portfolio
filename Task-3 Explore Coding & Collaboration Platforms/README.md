# Task 3: Coding and Collaboration Platforms #

# Platforms Used:
For this task, I explored HackerRank for coding practice and Google Workspace (Forms and Sheets) for digital collaboration. These tools are essential for a Computer Science student to build technical logic and manage data effectively.
# Project Overview:
In Part A, I used HackerRank to solve the "Python Loops" challenge. This exercise required me to write a script that calculates and prints the square of non-negative integers. Successfully passing the test cases improved my understanding of Python syntax and algorithmic thinking. In Part B, I developed a "Digital Literacy Awareness Quiz" using Google Forms. This included a variety of question types like multiple-choice and short-answer to engage batchmates. I then linked this form to a Google Sheet to demonstrate how to automate data collection and organize responses in real-time.
# Academic Impact:
Practicing on HackerRank prepares me for technical interviews and competitive programming, which are vital for my specialization in Health Informatics. Meanwhile, mastering Google Workspace tools allows me to collaborate on research projects and create structured surveys for health data collection. These platforms empower me to be a more effective Digital Ambassador by providing the tools to teach and lead others in a digital-first academic environment.
